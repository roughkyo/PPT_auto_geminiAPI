# 🚀 빠른 시작 가이드

Google Gemini API를 사용하여 5분 안에 PPT를 생성하세요!

## ⚡ 3단계로 시작하기

### 1️⃣ API 키 발급 (2분)

1. [Google AI Studio](https://makersuite.google.com/app/apikey) 접속
2. **"Create API Key"** 클릭
3. API 키 복사 (형식: `AIzaSy...`)

### 2️⃣ API 키 설정 (1분)

**PowerShell에서 실행:**

```powershell
# .env 파일 생성
Copy-Item .env.example .env

# 메모장으로 열기
notepad .env
```

**메모장에서 수정:**

```env
GEMINI_API_KEY=여기에_발급받은_API_키_붙여넣기
```

저장 후 닫기 (Ctrl+S)

### 3️⃣ PPT 생성 (2분)

```powershell
# 설정 테스트
python test_api_setup.py

# PPT 생성
python generate_ppt.py
```

**실행 후 선택:**
- 모드: `2` (Gemini API로 새로운 슬라이드 생성)
- 주제: `양자 컴퓨팅의 미래`
- 슬라이드 개수: `5`

완료! 🎉 `output/` 폴더에서 생성된 PPT를 확인하세요.

## ❓ 자주 묻는 질문

**Q: API 키는 무료인가요?**
A: 네! 무료 티어로 월 1,500회 요청 가능합니다.

**Q: 생성된 PPT를 수정할 수 있나요?**

A: 현재는 이미지 프롬프트만 생성됩니다. 이미지는 별도로 추가해야 합니다.
